Competency Highlights

Software Engineering & Design

The application provides a login screen backed by SQLite, stores and updates inventory items locally, shows the inventory in a grid layout, supports add edit delete actions with input validation, and sends low stock SMS notifications after the user grants permission.

Algorithms & Data Structures

The application loads course data from a CSV file, stores courses in a Binary Search Tree (BST) keyed by course ID, prints an alphanumeric sorted list of courses via in-order traversal, and displays details (title and prerequisites) for a selected course.

Databases & Visualization

The application loads data from MongoDB collections, authenticates users, exposes create read update and delete operations through a dashboard interface, supports filtering and search for larger data sets, and presents results as interactive tables and charts built from aggregation queries.
